Criado a estrutura de pastas do projeto, para melhor entendimento e facilitar o desenvolvimento

Devido a deficiencia com o uso do Oracle, acabei tendo mais facilidade com o SQL Server, por ser um banco mais comum, encontrei mais materiais e 
por este motivo, preparei a aplicação para uso de dois bancos de dados, sendo que o banco efetivo, esta sendo o SQL Server.

Projetos:
- Domain - Minhas enntidades/espelhos das tabelas
- Repositorio - Projeto que faz o acesso a dados (ao banco de dados)
- API - Disponivel na internet para que qualquer device ou solução consiga utilizar meu serviço exposto
